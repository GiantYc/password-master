# Password Master - 密码生成器

## 项目概述
Password Master 是一个简单易用的在线密码生成工具，帮助用户创建安全、强壮的密码。

## 产品目标
- 为用户提供便捷的密码生成服务
- 确保生成的密码符合现代安全标准
- 提供友好的用户界面和使用体验

## 功能需求

### 1. 密码生成
- 可自定义密码长度（8-64位）
- 可选择包含的字符类型：
  - 大写字母 (A-Z)
  - 小写字母 (a-z)
  - 数字 (0-9)
  - 特殊字符 (!@#$%^&*等)
- 一键生成随机密码
- 一键复制生成的密码

### 2. 密码强度检测
- 实时显示密码强度等级
- 提供密码强度改进建议

### 3. 用户界面
- 响应式设计，支持移动端和桌面端
- 简洁清晰的操作界面
- 深色/浅色主题切换

### 4. 附加功能
- 密码生成历史记录（可选）
- 常用密码模板保存（可选）
- 密码强度评分说明

## 技术要求
- 前端：HTML5, CSS3, JavaScript
- 框架：React/Vue.js
- 响应式设计
- 本地运算，不传输密码数据

## 安全要求
- 所有密码生成过程在客户端完成
- 不存储用户生成的密码
- 使用安全的随机数生成算法

## 后续规划
- 支持更多密码生成规则
- 添加密码管理功能
- 提供浏览器插件版本
- 支持多语言界面
